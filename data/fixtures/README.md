# Test Fixtures

Standardized JSON data for testing the Agentic Trading Platform.

| File | Model | Description |
|------|-------|-------------|
| `sample_option_chain.json` | `OptionChainResponse` | snapshot of NIFTY and BANKNIFTY option chains with Greeks. |
| `sample_crypto_orderbook.json` | `CryptoOrderbook` | L2 orderbook for BTCUSDT. |
| `sample_feature_vector.json` | `FeatureVector` | Computed indicators used for strategy input. |
| `sample_signals.json` | `List[TradeSignal]` | Example buy/sell signals generated by strategies. |
| `sample_portfolio.json` | `PortfolioState` | Combined view of equity and crypto positions and margins. |
| `sample_risk_params.json` | `RiskParams` | Configuration for RiskEngine limits. |
| `sample_risk_decision.json` | `RiskDecision` | Snapshot of a risk check result. |
| `sample_strategy.json` | `StrategyCreate` | Configuration for creating a new strategy. |

## Usage
These fixtures should be loaded in tests using `json.load()` and validated against Pydantic models in `src.models`.

## Validation
To ensure all fixtures are up to date with the latest models, run:
```bash
python tests/validate_fixtures.py
```
